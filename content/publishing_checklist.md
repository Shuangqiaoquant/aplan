# Publishing Checklist

Use this before posting any APlan / ShuangqiaoQuant content.

## Compliance

- [ ] No personalized investment advice.
- [ ] No "buy", "sell", "hold", "target price", or "position" instruction for readers.
- [ ] No return promise, win-rate claim, or income implication.
- [ ] No fundraising, pooled trading, entrusted account management, or follow-trade language.
- [ ] No private token, account, broker, holding, or licensed raw data included.
- [ ] Disclaimer is present.

## Research Integrity

- [ ] Claims match current project state: `research_only`.
- [ ] Backtest or validation limits are stated when mentioned.
- [ ] Failure cases are not hidden.
- [ ] "Candidate" is clearly separated from "actionable signal".
- [ ] Agent role is described as evidence processing, not autonomous trading.

## Platform Fit

- [ ] WeChat: coherent long article with clear sections.
- [ ] Zhihu: answers a concrete question or invites technical discussion.
- [ ] Xiaohongshu: short, visual, and non-promissory.
- [ ] GitHub link or project name is included when appropriate.

## Feedback Capture

- [ ] End with one or two feedback questions.
- [ ] Save meaningful comments to `content/feedback/`.
- [ ] Convert repeat feedback into issues, docs tasks, or research backlog items.
